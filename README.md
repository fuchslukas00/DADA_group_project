# Estimation of forced climate change from historical climate simulations using ridge regression and boosted trees

Group project for the course Introduction to Advanced Data Analytics


## Set up environment

Set up the R kernel needed to run the Jupyter Notebook with the following commands in your terminal:

```bash
conda env create -f environment.yml
```

```bash
conda activate r-DADA-proj
```

```bash
Rscript -e 'IRkernel::installspec(name="r-DADA-proj", displayname="R 4.4.3 (r-DADA-proj)")'
```

Then you should be able to select the kernel in the Jupyter Notebook.
